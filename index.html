Bienvenidos a la Guía de Comandos Básicos de HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. A continuación, se presentan algunos comandos básicos utilizados en HTML.

  • <h1> a <h6>: Etiquetas de encabezado, donde <h1> es el más importante y <h6> el menos importante.
  • <p>: Define un párrafo.
  • <a href="">: Crea un enlace a otra página o sección.
  • <img src="" alt="">: Inserta una imagen.
  • <ul> y <li>: Crea una lista no ordenada.
  • <ol> y <li>: Crea una lista ordenada.
  • <br>: Inserta un salto de línea.
  • <hr>: Inserta una línea horizontal para separar contenido.
